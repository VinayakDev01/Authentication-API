### appsettings.json Documentation

#### JWT Configuration
- `Jwt.Key`: The secret key used for signing the JWT. Ensure this is a secure and secret key.

#### Logging Configuration
- `Logging.LogLevel.Default`: The default logging level.
- `Logging.LogLevel.Microsoft`: The logging level for the Microsoft namespace.
- `Logging.LogLevel.Microsoft.Hosting.Lifetime`: The logging level for the Microsoft.Hosting.Lifetime namespace.

#### Allowed Hosts Configuration
- `AllowedHosts`: Specifies which hosts are allowed to access the application. Use `*` to allow all hosts.
